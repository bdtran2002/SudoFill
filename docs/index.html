Signup without the inbox mess.

SudoFill combines temporary Mail.tm inboxes, fast signup autofill, and verification-link handling in one browser extension. It is built for short-lived accounts and quick, low-friction signups.

Highlights

Everything in the README, trimmed into a landing-page view.

Temporary inboxes

  • Mail.tm-based disposable mail
  • Create, refresh, discard, and copy flows
  • Local session state for the current inbox

Autofill

  • Name, email, birth date, address, and more
  • Optional generated passwords
  • Skips hidden, read-only, or already-filled fields

Verification flow

  • Mailbox UI for reading messages
  • Detected verification links and codes
  • Domain-aware popup on matching pages

How it works

A short path from signup page to verified account.

1

Open SudoFill

Start from the popup, side panel, or mailbox view.

2

Create a temp inbox

Generate a disposable email address for the signup.

3

Autofill the form

Fill standard registration fields on the active page.

4

Verify and finish

Open the link or code from the mailbox and complete signup.
