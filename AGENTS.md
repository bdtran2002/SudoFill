# AGENTS

- Always use `bun` instead of `npm` or `node`
- Always use a **Conventional Commit** message format for `git commit` messages
- Keep changes minimal and targeted.
- `background` is reserved for future email polling, badge, and notification work.
- `popup` is reserved for quick actions and inbox summary UI.
- `options` is reserved for user preferences.
- `content` is reserved for future page form-detection and fill behavior.
- Prefer error-as-values patterns in future feature code; `neverthrow` is installed for that purpose.
